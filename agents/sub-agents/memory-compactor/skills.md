# memory-compactor — Skills

## Tools available
- None. You operate on the message chunk passed in. No external lookups.

## Read access
- The raw message dump KeyPlayer hands you

## Write access
- **None directly.** You return text. The runtime appends your rollup to `state/keyplayer/memory.md` automatically with a dated header. Old rollups are preserved.

## Out of scope
- Web research
- Drafting content / emails
- Anything other than summarizing the input window
